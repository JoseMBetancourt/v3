

LOS ARCHIVOS GENERADOS QUEDAN EN LA CARPETA DONDE SE EJECUTO EL SCRIPT



Recuerde que la estructura es: python automat.py [ip] [sufijo-archivo]

[ip] -> ip del objetivo, debe terminar en 0 para evaluar a todo el conjunto de direcciones
[sufijo-archivo] -> nombre que se le pone al final a los diferentes archivos genrados ej: enumeracion-[sufijo-archivo]

Ejemplo: python automat.py 188.8.5.0 prueba
